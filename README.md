# 📊 Global COVID-19 Trends Dashboard (Tableau)

## 📌 Project Overview
This project presents an interactive dashboard analyzing global COVID-19 trends using Tableau.  
The dashboard visualizes confirmed cases, deaths, recoveries, and vaccination trends across countries and regions.

---

## 🎯 Objective
The objective of this project is to:
- Analyze global COVID-19 case trends
- Compare country-wise and region-wise statistics
- Track daily and cumulative case growth
- Identify patterns and peak outbreak periods
- Support data-driven insights through interactive visualizations

---

## 🛠 Tools & Technologies Used
- Tableau
- Tableau Public (for publishing)
- Excel / CSV (Data Source)
- Data Cleaning & Preparation

---

## 📊 Key Metrics Included
- Total Confirmed Cases
- Total Deaths
- Total Recoveries
- Active Cases
- Mortality Rate (%)
- Vaccination Rate (if included)

---

## 📈 Dashboard Analysis Includes
- Country-wise case comparison
- Daily new cases trend
- Cumulative cases growth
- Death rate analysis
- Regional distribution map
- Top 10 affected countries

---

## 🌍 Dashboard Features
- Interactive filters (Country, Region, Date)
- Map-based visualizations
- Dynamic trend analysis
- Drill-down capability
- Clean and informative layout
